## ğŸ’¡ Tips & Tricks

### Getting Better AI Content

1. **Be Specific**: "Machine Learning in Healthcare" > "ML"
2. **Choose Right Tone**: 
   - Professional: Business presentations
   - Academic: Research, education
   - Creative: Marketing, design
   - Casual: Team meetings
   - Persuasive: Sales, pitches

3. **Optimal Slide Count**: 5-7 slides for most topics

### Editing Efficiently

1. **Use Keyboard**: Tab between fields
2. **Keep It Concise**: 3-5 bullets per slide
3. **One Idea Per Slide**: Don't overcrowd
4. **Use Layouts**: Match layout to content type

### Exporting

**Use PDF when**:
- Sharing for viewing only
- Sending to clients
- Posting online
- Exact appearance matters

**Use PPTX when**:
- Need to edit in PowerPoint
- Collaborating with others
- Adding animations later
- Customizing further

## ğŸ”§ Troubleshooting

### Images Not Showing

**Problem**: Slides don't have images  
**Solution**:
1. Check Replicate API token is configured
2. Wait for image generation (takes 10-15 seconds)
3. Images appear on image-text layout slides
4. Background images on other layouts

### Can't Find Saved Presentation

**Problem**: Presentation disappeared  
**Solution**:
1. Check you're in the same browser
2. localStorage is browser-specific
3. Check browser didn't clear data
4. Look in "Your Presentations" section on home

### Export Not Working

**Problem**: PDF/PPTX won't download  
**Solution**:
1. Check browser allows downloads
2. Disable popup blockers
3. Try different format (PDF vs PPTX)
4. Check browser console for errors

### Slow Generation

**Problem**: Takes too long to generate  
**Solution**:
1. Normal time: 15-20 seconds
2. Reduce slide count for faster generation
3. Check internet connection
4. Verify API keys are valid
