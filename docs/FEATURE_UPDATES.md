## Technical Details:

### Storage:
- **Method**: localStorage (browser-based)
- **Key Format**: `presentation_{id}`
- **Data**: JSON with slides, topic, timestamps
- **Capacity**: ~5-10MB (hundreds of presentations)

### Export Formats:

**PDF**:
- Uses html2canvas for screenshots
- jsPDF for PDF generation
- 1920x1080 resolution
- Landscape orientation

**PPTX**:
- Uses pptxgenjs library
- Native PowerPoint format
- 16:9 aspect ratio
- Editable text and bullets
- Supports images and backgrounds

### Auto-Save:
- Triggers on slide changes
- Uses React useEffect hook
- Debounced to prevent excessive writes
- Includes timestamp for sorting
